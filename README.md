# Liquid Sphere Sketch

Interactive browser sphere with subtle cursor-driven distortion on a warm paper background, plus a simplified rotating terminal sphere.

## Browser

Open `index.html` in a browser to view the original canvas version.

## Terminal

Run the additional terminal renderer with:

```bash
node terminal-sphere.js
```

Useful flags:

- `--frames 120` to stop automatically after a fixed number of frames
- `--fps 18` to slow the refresh rate down
- `--no-color` for monochrome output
- `--no-label` to remove the footer
